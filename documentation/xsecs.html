tt+DM reinterpretation, x-sec investigation¶

Are decays of the mediator $\phi$ into $t\bar{t}$ for masses above the tt threshold accounted for? There is no visible kink in the limit curve that is making us suspicuous.

Calculating the x-secs¶

First, calculate x-secs with similar settings as used for the central CMS samples. The MG process looks like this:

import model DMsimp_s_spin0
define p = g u c d s b u~ c~ d~ s~ b~
generate p p > xd xd~ t t~ / a z w+ w- @0

Here, I don't include an extra parton in the ME to save some time. The model comes from here: CMS extra model repository

Couplings and masses are then edited to

set param_card MASS 6 172.5
set param_card DECAY 6 auto
set param_card YUKAWA 6 172.5
set param_card MASS  52 1
set param_card MASS  54 500
set param_card DECAY 54 auto
set param_card gSXd  1.0
set param_card gPXd  0.0
set param_card gSd11 1.0
set param_card gSu11 1.0
set param_card gSd22 1.0
set param_card gSu22 1.0
set param_card gSd33 1.0
set param_card gSu33 1.0
set param_card gPd11 0.0
set param_card gPu11 0.0
set param_card gPd22 0.0
set param_card gPu22 0.0
set param_card gPd33 0.0
set param_card gPu33 0.0
set param_card gSg   0.0
set param_card gSh1  0.0
set param_card gSb   0.0
set param_card gSw   0.0
set param_card gPg   0.0
set param_card gPb   0.0
set param_card gPw   0.0

Particle 54 corresponds to the mediator, and this mass is scanned. The resulting x-secs are given in xsec_inv.

Note to myself: One needs to be careful that the widths are actually recomputed, which is apparently not done if you just use "launch" again and change one parameter in MG.

Now I compute the inclusive cross section, $p p \to t \bar{t} \phi$. The mediator is forced to be on-shell here, but that should be fine for this simple check. The results are given in xsec_incl.

One can calculate the ratio $\Gamma_{ratio} = \frac{\Gamma(\phi\to t \bar{t})}{\Gamma(\phi\to \chi \chi)}$ following https://arxiv.org/pdf/1605.09242.pdf

In [5]:
def width_ratio(m_med):
    return (3/2)*((m_med**2-4*172.5**2)/(m_med**2))**(3/2)

Results¶

We can compare the x-sec we get from MG directly with the one we get when calculating the inclusive x-sec and the respective branching fraction. From the results we can conclude that MG correctly takes the decay $\phi\to t \bar{t}$ into account.
